# ESP32 + L298N 手機 WiFi 遙控車

ESP32 開啟 WiFi AP 熱點，手機連上後用瀏覽器控制兩顆直流馬達。
不需要安裝任何 App。

## 硬體
- ESP32 開發板（你的接在 `/dev/cu.usbserial-120`）
- L298N 馬達驅動板
- 兩顆直流馬達 + 車體
- 電池組（建議 6~9V，給 L298N 的 VCC 用）

## 接線

| L298N | ESP32 GPIO | 說明 |
|-------|------------|------|
| IN1   | 14         | 左馬達方向 A |
| IN2   | 27         | 左馬達方向 B |
| IN3   | 26         | 右馬達方向 A |
| IN4   | 25         | 右馬達方向 B |
| ENA   | 32         | 左馬達 PWM 調速 |
| ENB   | 33         | 右馬達 PWM 調速 |
| GND   | GND        | **共地，必接** |
| VCC   | 電池正極    | 馬達電源（**不要**接 ESP32 5V） |

> ESP32 用 USB 或獨立 5V 供電，不要從 L298N 的 5V Out 拉電（電流不夠且雜訊大）。

## 燒錄
1. Arduino IDE → 工具 → 開發板 → 選擇 ESP32 對應型號（常見：`ESP32 Dev Module`）
2. 連接埠選 `/dev/cu.usbserial-120`
3. 開啟 `esp32-wifi-car.ino` → 上傳

## 使用
1. 開電後序列埠監控應顯示：`AP IP: 192.168.4.1`
2. 手機 WiFi 連 **ESP32-Car**，密碼 **12345678**
3. 瀏覽器開 `http://192.168.4.1`
4. 按鈕：▲ 前進、▼ 後退、◀ 左轉、▶ 右轉、■ 停止；下方拉桿調速

## 除錯
- 馬達不轉 → 檢查共地、電池電壓、L298N 跳線（ENA/ENB 跳帽要拔掉才能用 PWM）
- 方向反了 → 把 IN1/IN2（或 IN3/IN4）對調，或在程式裡把該邊的 `motorSpeed` 改負號
- 一邊強一邊弱 → 兩顆馬達特性差異，可在 `forward()` 給兩邊不同 speed 補償
