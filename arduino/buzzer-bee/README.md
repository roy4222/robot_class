# 小蜜蜂 蜂鳴器演奏

## 接線圖

```
Pin 8 ──→ 蜂鳴器 (+) ──→ 蜂鳴器 (-) ──→ GND
```

在 Tinkercad 上：
1. 搜尋 "Piezo" 放一個蜂鳴器
2. 正極接 Pin 8，負極接 GND

## 核心觀念：tone()

```cpp
tone(pin, frequency);           // 開始發出指定頻率的聲音
tone(pin, frequency, duration); // 發聲指定時間後自動停
noTone(pin);                    // 停止發聲
```

頻率 = 音高，常用音階對照：

| 音名 | Do | Re | Mi | Fa | Sol | La | Si |
|------|-----|-----|-----|-----|------|-----|-----|
| 頻率 | 262 | 294 | 330 | 349 | 392 | 440 | 494 |

## 可以改的地方

| 改什麼 | 怎麼改 |
|--------|--------|
| 速度 | `tempo` 改小更快，改大更慢 |
| 換歌 | 修改 `melody[]` 和 `beats[]` 陣列 |
| 音量 | 硬體上串電阻可以降低音量 |
