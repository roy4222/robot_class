## 零件筆記

### 1. 電位器（可變電阻 / 電位計）
- 用途：調整大小（音量、亮度）
- 腳位：A0（類比輸入，10-bit 解析度，0~1023）
- 實作：搭配 D9–D11 的 RGB LED，做亮度調整（用 PWM 模擬調光）
- 原理：電阻值隨旋轉改變 → 電位改變 → 類比讀值改變

### 2. DHT11 溫濕度感測器
- 腳位：D4
- 範圍：溫度 0–50°C、濕度 20–90% RH
- 需要 Library：**Adafruit DHT sensor library**
  - 安裝路徑：草稿碼 → 管理程式庫 → 搜尋「DHT」→ Adafruit 版本
  - 安裝時一併安裝相依的 Adafruit Unified Sensor
- 注意：每 2 秒讀一次（硬體限制）

### 3. LM35 半導體溫度計
- 腳位：A2（類比輸入）
- 範圍：約 -55°C ~ 150°C，反應快
- 不需要額外 library，直接用公式換算：
  ```
  電壓(V) = 讀值 / 1023.0 * 5.0
  溫度(°C) = 電壓 * 100
  ```
- 優點：反應速度快（直接貼在被測物表面）
- 缺點：無校正，讀值可能有偏差

### 4. 光敏電阻（光明電阻 / LDR）
- 腳位：A1
- 原理：光越強 → 電阻越小 → 類比讀值越大
- 應用：自動夜燈、防盜（光線突然出現觸發警報）
- 模擬器：Tinkercad 有此零件可練習

### 5. 紅外線接收器
- 腳位：D6
- 用途：接收遙控器訊號（如冷氣遙控器）
- 需要 Library：搜尋「IRremote」或類似名稱
- 限制：發射端與接收端必須「看得到彼此」才能通訊

## 感測器選型概念

| 用途 | 推薦零件 |
|------|----------|
| 環境溫濕度 | DHT11 / DHT22（精度更高） |
| 高溫測量（烤箱等） | 熱電偶（Thermocouple，可到幾百°C） |
| 電路板溫度監控 | LM35 / LM36（半導體型，IC 專用） |
| 環境亮度 | 光敏電阻（LDR） |

## 作業

### 內容
用 **Tinkercad 模擬器**做一個主題作品，自己發想一個想解決的問題或想做的效果。

### 使用零件
使用板子上已有的零件即可（**避免馬達類**，因為需要額外的馬達驅動板 L293D）

### 繳交格式：**ODF 格式**（非 Word .docx）
> 可用 OpenOffice / LibreOffice 開啟後另存為 ODF，或直接用 Word 另存為 .odt

繳交內容包含：
1. **主題說明** — 想做什麼？解決什麼問題？
2. **零件列表** — 用了哪些零件
3. **電路接線圖** — Tinkercad 截圖（或整個畫面截圖）
4. **程式碼** — 從 Tinkercad 複製貼上
5. **（加分）Tinkercad 分享連結** — 按右上角「分享」複製連結附上

### 靈感參考（可自由發揮）
- 門鈴（按鈕 + 蜂鳴器，按不同鈕唱不同歌）
- 防盜警報（光敏電阻偵測光線突變 → 蜂鳴器響）
- 自動夜燈（光敏電阻 + LED）
- 溫度警報（DHT11 + LED 紅藍燈）
- 記憶遊戲 Simon Says（兩個按鈕 + 蜂鳴器）
- 節拍器 / 換歌播放器
